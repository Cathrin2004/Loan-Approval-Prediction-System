# Loan Approval Prediction using Machine Learning

This is a mini Data Science and Machine Learning project that predicts whether a loan application will be approved or not using Logistic Regression.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data preprocessing
- Missing value handling
- Categorical data conversion
- Machine learning model training
- Loan approval prediction
- Accuracy evaluation
- Confusion matrix
- Classification report
- Data visualization

## Dataset
Dataset contains:
- Gender
- Married
- Dependents
- Education
- Self_Employed
- ApplicantIncome
- CoapplicantIncome
- LoanAmount
- Credit_History
- Property_Area
- Loan_Status

## Machine Learning Algorithm
- Logistic Regression

## Accuracy
The model achieved approximately 79% accuracy.

## Outputs
- Loan approval count graph
- Confusion matrix
- Classification report

## Project Structure
- data/ → dataset
- src/ → Python source code
- outputs/ → saved graphs

## Objective
To predict whether a loan application will be approved based on applicant details using Machine Learning techniques.
